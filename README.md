PolibVent - Sistem Manajemen Event Kampus
PolibVent adalah platform manajemen event untuk Politeknik Negeri Batam yang memungkinkan admin dan pengguna untuk mengelola berbagai kegiatan kampus secara digital.

Fitur Utama
Untuk Admin
Dashboard Admin: Antarmuka lengkap untuk mengelola semua event

Tambah Event Baru: Form dengan validasi lengkap dan preview poster

Edit/Update Event: Modifikasi event yang sudah ada

Persetujuan Event: Sistem approval (Disetujui/Menunggu/Ditolak)

Pencarian Event: Fitur pencarian real-time

Status Event: Aktif/Nonaktif dengan badge visual

Untuk Pengguna Umum
Beranda: Tampilan semua event dengan grid responsif

Detail Event: Halaman khusus dengan informasi lengkap

Edit Event (User): Kemampuan mengedit event dengan reset status persetujuan

Pencarian: Filter event berdasarkan kata kunci

Tambah Event: User dapat mengajukan event baru

Fitur Teknis
Database Integration: API PHP dengan MySQL

LocalStorage Fallback: Offline capability

Validasi Form: Client-side validation dengan feedback visual

Upload File: Poster/gambar dengan validasi (max 5MB, format JPG/PNG/WebP)

Responsive Design: Kompatibel dengan semua perangkat
